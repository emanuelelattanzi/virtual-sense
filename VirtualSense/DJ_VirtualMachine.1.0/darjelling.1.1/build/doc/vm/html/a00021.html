static void NEWARRAY ( ) [inline, static]

Executes the NEWARRAY instruction. The array type is fetched from code (uint8_t) and the length is popped from the stack. Note that only types T_BOOLEAN, T_BYTE, T_SHORT and T_INT are currently supported. An array of references is created with the ANEWARRAY instruction.
