Detailed Description

A two-byte typle that references entities. The infusion_id points to an infusion and the entity_id indexes a certain entity within that infusion. The local ID is called 'local' because it only makes sense within the context of the infusion in which it is found. This is because the infusion_id indexes into an import list in the infusion. Local IDs can be resolved into global IDs.
